# NutriFlow

NutriFlow is a modern meal planning and nutrition website built with HTML, CSS, and JavaScript.

The platform helps users discover healthy meal ideas, organize weekly meal plans, explore nutrition insights, and watch recipe preparation videos directly on the website.

---

## Features

### Modern Responsive UI
- Clean SaaS-inspired design
- Responsive layout for desktop, tablet, and mobile
- Smooth transitions and hover effects
- React-inspired interface styling

### Smart Meal Planning
- Weekly meal planner
- Structured meal organization
- Balanced nutrition suggestions

### Recipe Discovery
- Multiple healthy meal categories
- High protein meals
- Vegetarian meals
- Quick meals
- African dishes

### Interactive Features
- Smooth dropdown navigation
- Mobile responsive menu
- Scroll reveal animations
- Recipe category filtering
- Embedded cooking videos

### Nutrition & Wellness
- Nutrition guidance section
- Medical recommendation references
- Trusted health resource links
